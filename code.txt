git init 初始化项目
git --version 查看git版本
git config --global user.name "Your Name" 用户名
git config --global user.emaik "wvgyweg@qq.com" 邮箱

git add. 添加到暂存盘
git checkout 文件名 从暂存盘还原文件

git commit -m “提示文本” 提示到版本库

git log 查看版本库历史
git reset --hard 哈希值 当前版本重新指向哪个版本
git reflog 查看历史版本
git reset --hard HEAD^ 回退到上一个版本 可以100^
git checkout -b dev 创建一个名为dev的分支
git checkout master 切换到master分支
git branch mumu 创建一个mumu的分支
git branch 查看分支
git merge dev 把dev分支合并到当前分支
